### 技术工具

- UML建模工具：Plantuml

### 如何使用

1. 克隆仓库：`git clone https://github.com/wanrenhuifu/JLU_UML.git`
2. 查看各目录下的建模文档和图表
3. 参考`大作业题目.md`了解作业要求
4. 参考`大作业模板.md`了解文档格式
